# Kiai.wtf

Kiai.wtf is our leaderboard site that provides leaderboards for all guilds that Kiai is in.

## URL Format

Most leaderboards hosted at the following address:
```
https://kiai.wtf/guild/608098293498511360
```
where 608098293498511360 is the ID of the server you are viewing the leaderboard for.

Premium guilds can have a vanity link, such as this:
```
https://kiai.wtf/kiaisupport
```
