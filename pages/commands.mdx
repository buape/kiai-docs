# Command Comparison

## This table gives you the low-down on all the commands of Kiai and their properties.

| Command                                            |                                    Description                                    | Required Permissions | Guild Only | Premium |
| :------------------------------------------------- | :-------------------------------------------------------------------------------: | -------------------: | ---------- | ------- |
| [`about`](/commands/about)                         |                          Get information about the bot.                           |                    - | ✅         | ❌      |
| [`application`](/commands/application)             |          Authorize and deauthorize external applications in your server.          |         Manage Guild | ✅         | ❌      |
| [`blacklist`](/commands/blacklist)                 |                       Manage the XP user or role blacklist.                       |         Manage Guild | ✅         | ❌      |
| [`custom-formula`](/commands/custom-formula)       |                   Manage the custom formula for calculating XP.                   |         Manage Guild | ✅         | ❌      |
| [`export`](/commands/export)                       |                              Export data from Kiai.                               |         Manage Guild | ✅         | ❌      |
| [`import`](/commands/import)                       |                         Import XP data from another bot.                          |         Manage Guild | ✅         | ❌      |
| [`leaderboard`](/commands/leaderboard)             |                       View the leaderboard for the server.                        |                    - | ✅         | ❌      |
| [`leaderboard-roles`](/commands/leaderboard-roles) | Manage the roles members get when reaching a certain position on the leaderboard. |         Manage Guild | ✅         | ❌      |
| [`level-up`](/commands/level-up)                   |                 Manage the settings for level ups in this server.                 |         Manage Guild | ✅         | ❌      |
| [`multipliers`](/commands/multipliers)             |                                Manage multipliers                                 |         Manage Guild | ✅         | ❌      |
| [`ping`](/commands/ping)                           |                             Check the bot's latency.                              |                    - | ✅         | ❌      |
| [`premium`](/commands/premium)                     |                               Manage your premium.                                |                    - | ✅         | ❌      |
| [`rank`](/commands/rank)                           |                             Check the rank of a user.                             |                    - | ✅         | ❌      |
| [`reset`](/commands/reset)                         |                  Resets all settings and levels for this server.                  |          Guild Owner | ✅         | ❌      |
| [`rewards`](/commands/rewards)                     |                        Manage the rewards on this server.                         |         Manage Guild | ✅         | ❌      |
| [`settings`](/commands/settings)                   |                         Manage the settings for the bot.                          |         Manage Guild | ✅         | ❌      |
| [`xp`](/commands/xp)                               |                    Manage the XP of a user or users in a role.                    |         Manage Guild | ✅         | ❌      |
