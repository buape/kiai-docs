# Command Comparison

## This table gives you the low-down on all the commands of Kiai and their properties.

| Command             |                                    Description                                    | Required Permissions | Guild Only | Premium |
| :------------------ | :-------------------------------------------------------------------------------: | -------------------: | ---------- | ------- |
| `about`             |                          Get information about the bot.                           |                    - | ✅         | ❌      |
| `application`       |          Authorize and deauthorize external applications in your server.          |         Manage Guild | ✅         | ❌      |
| `blacklist`         |                       Manage the XP user or role blacklist.                       |         Manage Guild | ✅         | ❌      |
| `debug`             |                    Generate a debug report for the Kiai staff.                    |                    - | ✅         | ❌      |
| `export`            |                              Export data from Kiai.                               |         Manage Guild | ✅         | ❌      |
| `factory-reset`     |                  Resets all settings and levels for this server.                  |          Guild Owner | ✅         | ❌      |
| `import`            |                         Import XP data from another bot.                          |         Manage Guild | ✅         | ❌      |
| `leaderboard`       |                       View the leaderboard for the server.                        |                    - | ✅         | ❌      |
| `leaderboard-roles` | Manage the roles members get when reaching a certain position on the leaderboard. |         Manage Guild | ✅         | ❌      |
| `levelup`           |                 Manage the settings for level ups in this server.                 |         Manage Guild | ✅         | ❌      |
| `multipliers`       |                                Manage multipliers                                 |         Manage Guild | ✅         | ❌      |
| `ping`              |                             Check the bot's latency.                              |                    - | ✅         | ❌      |
| `premium`           |                               Manage your premium.                                |                    - | ✅         | ❌      |
| `rank`              |                             Check the rank of a user.                             |                    - | ✅         | ❌      |
| `rewards`           |                        Manage the rewards on this server.                         |         Manage Guild | ✅         | ❌      |
| `settings`          |                         Manage the settings for the bot.                          |         Manage Guild | ✅         | ❌      |
| `xp`                |                    Manage the XP of a user or users in a role.                    |         Manage Guild | ✅         | ❌      |
