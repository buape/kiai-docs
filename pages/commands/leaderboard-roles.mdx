# Leaderboard Roles Command

## Syntax

```
/leaderboard-roles add <role> <position>
/leaderboard-roles remove <reward>
/leaderboard-roles list
/leaderboard-roles sync
```

## Permissions

This command is available to users with the Manage Server permission.

## Description

The `/leaderboard-roles` command allows you to customize what roles members get when reaching a certain position on the leaderboard.
