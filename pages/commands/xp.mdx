# XP Command

## Syntax

```
/xp add <xp|level> [user] [role]
/xp remove <xp|level> [user] [role]
/xp prune [user]
/xp reset <user|role>
/xp set <xp|level> [user] [role]
```

## Permissions

This command is available to users with the Manage Server permission.

## Description

The XP command allows you to add or remove XP from users or roles. You can also specify a level to set the user or role to.

## Examples

```
/xp add xp:100 user:@john

/xp remove level:10 role:@Members

/xp prune

/xp reset user:@john

/xp set level:100 @Members
```
