# Application Command

## Syntax

```
/application authorize <id>
/application deauthorize <id>
/application list
```

## Permissions

This command is available to users with the Manage Server permission.

## Description

This command allows you to authorize or deauthorize a third-party application to access your server's data in Kiai.
<Callout emoji="⚠️">
	Be careful when giving third-party apps access to your Kiai data. Only use apps and services you trust. Sharing your data can put your server and member information at risk.
</Callout>
