# Reset Command

## Syntax

```
/reset all
/reset xp-only
```

## Permissions

This command is available to only the owner of the server.

## Description

Depending on the subcommand, this command will reset all data or only the XP data of all users in the server. This cannot be undone.

## Notes

You are required to use `/reset xp-only` before you can import any data into the server.
