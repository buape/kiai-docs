# Debug Command

## Syntax

```
/debug
```

## Permissions

This command is available to users with the Manage Server permission.

## Description

The `/debug` command generates a report with debug information for our team to use in troubleshooting. This report includes the following information:

-   Server Information such as the server ID, the bot's permissions, and the server's settings
-   Information about you, such as your Discord ID, permissions in the server, and level
-   General bot information such as the ping and version
