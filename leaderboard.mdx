Kiai features a way to set a automatically updating leaderboards that can be created using `/auto-leaderboard create`. 

Leaderboards options are: 
 - XP and Levels
 - Messages Sent
 - Voice Minutes
 - XP Streaks

 You can also set time and description parameters to customize your auto-leaderboards even more!
